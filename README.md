# Codex Skills Private

This private repository stores local Codex skills.

## Skills

- `codex-project-orchestrator`: project control workflow for PRD-to-milestone planning, task boards, thread handoffs, and integration logs.
- `implementation-planner`: turns technical plans into execution phases, task breakdowns, acceptance criteria, and validation plans.
- `liu-ying-chat-style`: private relationship/chat style skill. Keep this repository private.
- `product-co-creator`: turns a chosen product idea into concrete product direction, MVP scope, flows, requirements, and roadmap.
- `root-cause-fixer`: investigates bugs by reproducing, proving root cause, fixing completely, and validating.
- `technical-solution-designer`: turns PRD/product plans into technical architecture, data models, APIs, modules, and risks.

## Layout

```text
skills/
  codex-project-orchestrator/
  implementation-planner/
  liu-ying-chat-style/
  product-co-creator/
  root-cause-fixer/
  technical-solution-designer/
```

## Restore

Copy a skill directory into:

```text
~/.codex/skills/
```

Then restart or refresh Codex if the skill list does not update immediately.
