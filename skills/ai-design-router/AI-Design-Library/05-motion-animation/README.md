# Motion Animation

Use for animation, scroll effects, timeline choreography, transitions, and motion performance.

## Installed Or Plugin Skills

- `gsap-core`
- `gsap-timeline`
- `gsap-scrolltrigger`
- `gsap-react`
- `gsap-frameworks`
- `gsap-plugins`
- `gsap-performance`
- `gsap-utils`
- `figma:figma-implement-motion`
- `hyperframes:gsap`

## Default Choice

- Use `gsap-core` for basic GSAP usage.
- Use `gsap-timeline` for sequencing.
- Use `gsap-scrolltrigger` for scroll-linked animation.
- Use `gsap-react` for React/Next.
- Use `gsap-frameworks` for Vue/Svelte/other frameworks.
- Use `gsap-performance` when optimizing jank or heavy animation.

## Do Not

- Do not add GSAP when CSS transitions or framework-native motion are enough.
- Mix multiple animation systems in the same component tree without a clear reason.
- Skip reduced-motion handling for prominent motion.
