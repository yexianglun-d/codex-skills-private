# Frontend Implementation

Use for turning visual intent into code while preserving the stack and project conventions.

## Installed Or Plugin Skills

- `frontend-design`
- `build-web-apps:frontend-app-builder`
- `build-web-apps:react-best-practices`
- `vue-code-discipline`
- `build-web-apps:shadcn`
- `build-ios-apps:swiftui-ui-patterns`
- `build-ios-apps:swiftui-liquid-glass`
- `game-studio:game-ui-frontend`

## Default Choice

- Existing Vue project: `vue-code-discipline` first; add `frontend-design` only for visual work.
- React/Next: React best-practices plus `frontend-design` when visual design is central.
- New greenfield frontend app: `build-web-apps:frontend-app-builder` or `frontend-design`, depending on available plugin context.
- iOS SwiftUI: SwiftUI skill group.
- Browser game UI: game UI skill group.

## Do Not

- Change stack to match a skill.
- Add a design library without checking project dependencies.
- Use visual design skills for non-visual frontend bugs.
