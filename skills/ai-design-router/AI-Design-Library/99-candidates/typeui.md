# TypeUI Candidate

Repository: `bergside/typeui`

Current status: candidate, not installed.

TypeUI appears to provide UI design principles, prompt resources, CLI, MCP, and skill-style material. It overlaps with:

- `frontend-design`
- `taste-skill`
- `ui-ux-pro-max`

Recommendation:

- Do not install by default.
- Reconsider only if the current stack lacks reusable UI prompt patterns or if the existing taste/design skills fail to cover a repeated design need.
- `VoltAgent/awesome-design-md` is currently preferred as the lightweight design-reference library because it adds `DESIGN.md` reference files without adding another runtime, MCP, CLI, or design skill.

If installed later, keep it as a reference or fundamentals layer, not as another default visual design skill.
