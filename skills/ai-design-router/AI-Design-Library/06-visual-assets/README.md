# Visual Assets

Use for generated or transformed visuals, character art, spritesheets, and asset preparation.

## Installed Or Plugin Skills

- `hatch-pet`
- `game-studio:sprite-pipeline`
- `game-studio:web-3d-asset-pipeline`
- `figma:figma-generate-design`

## Catalog-Only Candidates

These entries are recorded for future review only. They are not installed, vendored, or selected by the default router.

- Article/editorial illustrations: `99-candidates/visual-content.md` -> `editorial-illustrations`
- Xiaohongshu/social cards: `99-candidates/visual-content.md` -> `xhs-visual-cards`

## Default Choice

- Use `hatch-pet` for Codex pet or character sprite workflows.
- Use game-studio asset skills for browser-game sprite or 3D asset pipelines.
- Use Figma generation skills for design assets that should live in Figma.

## Do Not

- Generate decorative assets when real product screenshots or existing brand assets are needed.
- Treat asset generation as implementation validation.
