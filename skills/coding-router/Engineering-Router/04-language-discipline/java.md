# Java Route

Use for Java, Spring, Spring Boot, MyBatis, Maven, Gradle, service-layer, transaction, exception, logging, database, DTO, entity, controller, or mapper work.

## Route

- Use `java-code-discipline`.
- Add `root-cause-fixer` for bugs, failing tests, regressions, or incorrect runtime behavior.
- Use Context7 for current framework, library, SDK, API, or CLI details.

## Focus

- Follow project structure and existing response, exception, logging, transaction, and validation conventions.
- Apply Alibaba Java manual principles through the compact `java-code-discipline` workflow.
- Add concise Chinese comments only for complex business logic or hard-to-read core decisions.

## Do Not

- Copy a large Java manual into the response.
- Add generic utility classes when project utilities already exist.
- Patch bad inputs with default values instead of fixing the source.
