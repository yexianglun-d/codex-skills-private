# Inbox Archive

主线程审核过的 `inbox/thread-updates/` 上报移动到这里。

每份归档必须包含：

- `Task ID`
- `Review Status: accepted`、`partially-accepted` 或 `rejected`
- `Reviewed By`
- `Reviewed At`
- 归档说明，说明哪些事实已合并，哪些被拒绝或转入 open questions

`VERIFIED` / `DONE` 任务必须能在这里找到 `Review Status: accepted` 的对应 Task ID。

不要把未审核的 worker 判断直接写入核心项目记忆。
