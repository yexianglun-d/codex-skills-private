## 记录要求

- 不能把“代码已写”当作“功能已完成”。
- `VERIFIED` / `DONE` 任务必须有对应 Task ID 的验证记录。
- 无法运行验证时，写明原因、替代检查和剩余风险。
- 涉及真实外部服务时，mock 只能说明本地逻辑，不代表真实接入完成。
