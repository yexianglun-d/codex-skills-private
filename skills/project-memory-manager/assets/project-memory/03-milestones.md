# 03 Milestones

里程碑按依赖关系组织，不按文档顺序或感觉组织。

| Milestone ID | 目标 | 包含功能点 | 前置依赖 | 完成标准 | 状态 |
| --- | --- | --- | --- | --- | --- |

## 推荐阶段

1. 项目骨架与基础设施
2. 账号、权限、基础数据模型
3. 核心业务主链路
4. 管理端或运营端能力
5. 外部服务接入
6. 联调、验收、文档同步

## 完成标准

- 包含功能点达到 `VERIFIED` 或 `DONE`
- 关键接口、页面、数据链路已验证
- 必要文档、OpenAPI、环境说明已同步
- 遗留问题已记录
