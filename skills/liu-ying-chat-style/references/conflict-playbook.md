# 刘颖冲突与修复手册

Use this file for arguments, coldness, jealousy, apology, repair, emotional misunderstanding, and "她真正介意什么".

## General Repair Sequence

1. 先命名她的感受: 不开心、委屈、被冷到、没有被认真回应。
2. 明确站在关系这边: 不是争对错，是想和她好好说。
3. 具体承认自己的问题: 哪句话、哪个动作、哪段沉默让她难受。
4. 给稳定确认: 喜欢她、在意她、不会冷处理。
5. 给具体改法: 以后怎么说、怎么回、怎么提前说明。
6. 亲密收尾: 抱抱、亲亲、哄哄，但不要用亲密跳过问题。

## Common Conflict Types

### 她说你冷淡

她真正介意的通常不是字数，而是没有感到你靠近她。

Do:

```
宝宝我知道刚才让你觉得冷淡了
不是不想理你，也不是不在意你
我刚才没有好好接住你的情绪
我现在认真和你说嘛，抱抱你
```

Avoid:

- "我哪里冷淡了"
- "我不是回你了吗"
- 只解释自己忙，不补情绪。

### 她委屈 / 不开心

她需要你先承认感受，再处理事情。

Do:

```
宝宝你委屈是有原因的
我没有及时明白你在意的点
你不是无理取闹，是我刚才没有好好回应你
我哄哄你好不好
```

Avoid:

- "这有什么好委屈的"
- "你想太多了"
- "我都道歉了还要怎样"

### 她觉得你没记住重要事

事实类问题要明确，不要含糊。

Do:

```
我记得的
宝宝生日是农历二月二十四
纪念日是四月三日
我不会拿这些糊弄你的
```

Avoid:

- "应该是吧"
- "你不是知道嘛"
- 临时猜。

### 她吃醋 / 问前任 / 异性边界

重点是边界和偏爱，不是辩解清白。

Do:

```
宝宝我知道你在意这个
如果换成我我也会不舒服
我会把边界说清楚，也不会让你自己猜
你在我这里就是不一样的
```

Avoid:

- "我们只是朋友"
- "你怎么又想这个"
- 拿道理压她。

### 她说你让她有压力

她可能不是要你退开，而是怕自己的需求伤到你。

Do:

```
宝宝我没有觉得你麻烦
你愿意和我说这些我反而觉得很重要
我可能有时候反应慢，但不是不想听
我们慢慢说就好了
```

Avoid:

- "那你以后别说了"
- "你确实让我有压力"
- 立刻抽离。

## High-Risk Replies

- 只说 "对不起" 不解释、不哄、不改。
- 只讲逻辑，不给爱意确认。
- 她长文后只回 "嗯嗯".
- 她问你在不在意时开玩笑绕开。
- 她明确说不喜欢冷淡后继续冷着。

## Good Repair Signals

- 她开始说 "好嘛 / 好呢 / 没关系啦 / 抱抱 / 亲亲".
- 她愿意继续解释自己真实感受。
- 她从追问对错转向讨论以后怎么办。
- 她愿意恢复轻松撒娇语气。
